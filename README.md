# PurchasePalace 🏰

A premium, responsive e-commerce web application designed to provide a seamless shopping experience for a wide variety of categories including fashion, electronics, home goods, and more.

## 🚀 Live Demo
[PurchasePalace](https://david-godspower.github.io/purchasepalace/)

## 🛠️ Tech Stack
* **HTML5**: Structured and semantic markup.
* **CSS3**: Responsive design featuring Flexbox, CSS Grid, and custom animations.
* **JavaScript (ES6+)**: Custom logic for cart management, real-time search filtering, and mobile navigation.
* **LocalStorage API**: Enables persistent shopping carts and user session management without needing a backend server.

## ✨ Key Features
* **Cinematic Hero Slider**: Automatically transitioning banners with custom fade-in animations and text overlays.
* **Mobile-First Navigation**: A fully responsive side-drawer "hamburger" menu for smooth mobile browsing.
* **Smart Search**: Real-time product filtering to help users find exactly what they need.
* **User Authentication**: Integrated Sign-in/Sign-up flow that persists user data.
* **Dynamic Shopping Cart**: Instant cart updates with badge count and order summary.
* **Professional UI/UX**: Clean aesthetic with hover effects, consistent branding, and intuitive layouts.

## 📂 Project Structure
* `/js/main.js`: Handles search, mobile menu, cart interactions, and slider logic.
* `/js/products.js`: Centralized inventory database for easy product updates.
* `/styles.css`: All site styling, including custom components and responsive media queries.
* `signin.html` / `signup.html`: Dedicated authentication interfaces.

## 💡 How to Run Locally
1. Clone this repository to your machine:
   `git clone https://github.com/your-username/PurchasePalace.git`
2. Open `index.html` in any modern web browser.
3. No backend setup is required!
